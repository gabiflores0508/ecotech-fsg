# 🌱 EcoTech — Plataforma de Conscientização Climática

Projeto desenvolvido como atividade de extensão da faculdade **FSG Centro Universitário**, com foco em conscientização ambiental e mudanças climáticas.

## 📋 Sobre o Projeto

O **EcoTech** é uma plataforma web interativa que tem como objetivo promover a conscientização sobre questões climáticas e ambientais. O site apresenta informações educativas sobre sustentabilidade, impactos das mudanças climáticas e ações que podem ser tomadas para contribuir com o meio ambiente.

## 🚀 Tecnologias

- HTML5
- CSS3
- JavaScript

## 🌐 Deploy

O site está hospedado na **Vercel** e pode ser acessado em:
